## Garantia e segurança

**Pague apenas depois de receber o primeiro projeto!**

Como meio de garantia de qualidade do meu trabalho e segurança ao contratante, o primeiro projeto realizado comigo é pago após a finalização do mesmo...

### Para agências

**Descontos de 10% no orçamento a cada um projeto confirmado comigo!**

Com a intenção de prestar serviço terceirizado para agências, dou desconto conforme a quantidade de projetos a se fazer, sendo 10% do valor por projeto.

Ou seja, a partir de dois projetos, a cada um projeto que for confirmado comigo, a agência recebe 10% de desconto no valor do orçamento de cada projeto individualmente.

## Vamos concluir uma demanda em 3 passos:

### Primeiro passo: Planejamento

Primeiro discutiremos a finalidade do projeto, após isso retornarei um orçamento fixo para realizar a demanda, fechando acordo com o contratante agrupo ou crio o material necessário para o próximo passo.

### Segundo passo: Execução

Estipulamos um prazo de entrega e inicio o desenvolvimento do projeto, atualizando o contratante em tempo real em relação ao progresso do projeto.

### Terceiro passo: entrega

Após realizar o projeto mando a primeira versão ao contratante para aprovação, caso aprovado, encerramos o projeto o mesmo é entregue e o pagamento é feito.
