## Desenvolvedor e Web designer

Apaixonado em criar **interações, motion graphics, artes gráficas, páginas e aplicações** com interface **intuitiva**, **minimalistas** e **elegantes**...

Alguns dos projetos que mais costumo entregar são:

- **Aplicações web**,
- **Artes gráficas**,
- **Páginas de vendas/conversão**,
- **Lojas online**,
- **Integrações de serviços**,
- Entre outros...

Ainda sim, realizo projetos simples e individuais como soluções ou criação de landing pages, abaixo segue algumas tecnologias usadas por mim que fortemente recomendo:

### TypeScript

Primeiro, a linguagem de programação. Estou muito acostumado a usar TypeScript, embora a criação de sites possa ser feita usando PHP, Python, low code e etc.

### React

Realmente gosto de construir aplicações web utilizando react, durante muito tempo foi, e é minha lib preferida.

### Vue.js

Quando se trata de landing pages, vue.js é uma mão na roda! Adoro utilizar esta tecnologia embora não tenha muita demanda.

### Tailwind CSS

Venho experienciando o uso de Tailwind CSS faz uns meses e adoro esta tecnologia, uso sempre que pertinente em meus projetos!.

### Framer Motion & spline

Para animações, Framer Motion é indispensável, costumo utilizar muito o Spline para lidar com interações em 3d, ambas tecnologias são incríveis e estou adorando utilizar!

### Next.js

Venho utilizando next.js pra quase todos os projetos que desenvolvi nos últimos meses, estou amando a tecnologia, facilita muitos aspectos do desenvolvimento.

## UI/UX Designer

Projetar interfaces de usuário é algo que sou realmente bom e tenho sido um designer de UI/UX desde 2022 até o presente

Estou sempre pensando em como criar a melhor experiência possível para o usuário, além de criar interfaces de usuário visualmente atraentes.

O maior desafio é encontrar o equilíbrio entre criar uma ótima interface de usuário e uma experiência de usuário incrível, já que estou sempre tentando inovar e me superar!
